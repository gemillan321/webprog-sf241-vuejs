1. PROJECT DESCRIPTION
----------------------
A professional, mobile-responsive portfolio website designed for a 
Cybersecurity and Digital Forensics student. This project utilizes 
Bootstrap 5 for layout, custom CSS for a "Cyber-Ops" aesthetic, 
and AOS for scroll-triggered animations.
